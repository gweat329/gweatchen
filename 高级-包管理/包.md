# 1. 模块
- 一个模块就是一个包含python代码的文件，后缀名称是.py就可以，模块就是python文件
- 随什么我们用模块
    - 程序太大，编写维护非常不方便，需要拆分
    - 模块可以增加代码重复利用的方式
    - 当做命名空间使用，避免命名冲突
- 如何定义模块
    - 模块就是一个普通文件，所以任何代码可以直接书写
    - 不过根据模块的规范，最好在模块中编写以下内容
        - 函数（单一功能）
        - 类（相似功能的组合，或者类似业务模块）
        - 测试代码
        
- 如何使用模块
    - 模块直接导入
        - 假如模块名称直接以数字开头,需要借助importlib帮助
    - 语法
        
        import module_name
        module_name.function_name
        module_name.class_name
